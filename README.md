# proyecto_sprint_7
Este es el repositorio del proyecto del Sprint 7 del curso de Ciencia de Datos de TripleTen. Elaborado por Gustavo Rivera
Este proyecto tiene por objetivo poner a prueba los conocimientos aprendidos en el Sprint 7, correspondiente a las herramientas de desarrollo de software.
La aplicación web desarrollada en este proyecto usa como base un dataset de anuncios de ventas de autos en los Estados Unidos.
La aplicación web despliega, a traves de casillas de verificación, el histograma de los odómetros del dataset o el diagrama de dispersión que relaciona el odometro del vehículo con su precio.
